# Control-plane read model unavailable

Use this runbook when the public summary or an authenticated Ramone tool
returns `503`.

1. Do not change OpenWebUI, Home Assistant, Ramone, or a provider credential.
2. Run the local fixture tests and confirm the committed fixture still passes
   leak and shape validation.
3. Check only whether the bounded KV key is present and when its producer last
   reported. Do not print the document if owner review has not confirmed its
   classification.
4. Validate a redacted copy locally. A missing, malformed, stale, or rejected
   source must remain non-healthy.
5. Repair the owning producer through a separate reviewed change. Do not add a
   generic fetch or provider proxy to this Worker.

Rollback the Phase 9 route group by reverting its focused repository change.
If OpenWebUI was enabled later, first disable only the Atlas external-tool
assignment. Existing Ramone tools and Home Assistant control remain untouched.
